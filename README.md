# 🩸 Blood Love Animation ❤️

A dark wine/red themed encrypted love animation built with pure:

- HTML

- JavaScript
- Canvas API

- 💻 Hacker-style loading screen
- ⌨️ Terminal typing animation
- 🔓 Decrypt message button
- ❤️ Animated heart made from love particles
- 🍷 Dark blood/wine color theme
- ✨ Neon red glow effects
- 💓 Heartbeat animation
- 📱 Responsive design
- 🚀 GitHub Pages ready
